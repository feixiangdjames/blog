    2. 常量

    • JAVA提供了两种变量类型：基本类型、引用类型。
    • JAVA定义基本类型：整（type,short,int,long）,浮点型（float,double）,boolean,char
    • 变量可以重新赋值；等号是赋值语句，不是数学符号。
    • 常量初始化之后，不能重新赋值；为了避免magic number. 常量修饰关键字是final

    关键字

    Java关键字是对Java编译器有特殊含义的字符串，是编译器和程序员的一个约定，程序员利用关键字来告诉编译器其声明的变量类型、类、方法特性等信息。
    声明变量要避开关键字。

    Hello程序结构

    在Java程序中，注释是一种给人阅读的文本，不是程序的一部分，所以编译器会自动忽略注释.在JAVA中有3种注释。

    1.// 单行注释
    2./~/多行注释
    3./*~/ 可以自动创建文档的注释。

    java 的基本单位是关键字class,一个.java 文件有且只有一个class
    class 的访问修饰符是public，表示可以在命令行运行
    public static void main 是JAVA程序的固定入口；参数必须是String[ ] args;方法定义了一组执行语句，方法内部的代码将会被依次顺序执行。main 不是关键字只是 JAVA的主函数入口。
    System.out.println 打印方法的执行，参数是”Hello,world”
    |关键字|说明|
    |–|–|
    |public||
    |static|    |
    |void|没有返回值||||
    方法代码每一行用；结束

    类的命名规则

    • class name 必须是英文字母，后接数字和下划线。
    • 常用惯例是声明类名称用大写字母开头，使用驼峰命名法。

      变量

    • 什么是变量？
      变量是对数据本身的引用。在java中会用变量名来指代。
    • 变量的行为？
      变量必须先定义，后使用。
      变量在同一个作用域（代码块中）不能被重复声明。
      变量定义后可以多次赋值。
      变量的命名是以小写字母开头后可以加数值；惯例上当多个单词组合成变量名使用驼峰命名法。

      数据类型

      java 变量分为两种基本类型和引用类型。

      基本类型

      存储在栈内存中，CPU可以直接进行运算的类型。
      计算机内存的最小存储单元是字节（byte），一个字节就是一个8位二进制数，即8个bit。二进制的表示是0000_0000~1111_1111.一个bit就是一个内存地址。
      不同的数据类型所占用的字节数是不同的。

      整型

      数据类型 字长 说明
      byte 1 整数类型,代表10进制的范围是-2^7~2^7-1,第一位是符号位，0是正，1是负；涉及到反码和补码知识
      short 2 整数类型
      int 4 整数类型
      long 8 整数类型

    不同进制下的数字是相同的，15==0xf==0b1111;

    浮点型

    浮点类型的数就是小数，整数也可以用小数表示的，使用科学计数法。
    定义小数类型的关键字是 float double
    |类型|字长(byte)|说明|
    |–|–|–|
    |float|4|浮点类型32bit|
    |double|8|长浮点类型，定义不需要末尾加f|

    • 不同于可以准确定义整数，小数是不准确，比如0.1+0.2===0.3 在不采用任何处理方式的情况下，结果是false.所以,在做小数结果的比较时候，要在一定精度范围内比较，才可以得出正确结果。

    字符型 char

    • 字符类型用单引号表示，只用有一个字符； 注意：字符和字符串不同；字符属于基本类型，字符串是引用类型。

    boolean

    • 只有两个值 ture 、false;
    • 通常表示比较的结果；

      常量

    • 如果变量用final修饰的话，就是常量；
    • 为了方便理解程序,用来代替Magic Number。

      常量初始化以后不能再次赋值。否则会报编译异常。
      常量名通常全部大写
      常量用来避免意外赋值
