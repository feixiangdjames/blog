    总结
    字符串是不可变对象
    字符串操作不改变原字符串内容，而是返回新字符串
    常用的字符串：提取子串，查找，替换，大小写，都不改变原有字符串
    字符串和 byte[]互相转换要注意编码，总是建议UTF-8

    #编码

    • 中文编码：GB212,GBK
    • 一个中文字符占2个字节
    • 第一个字节最高位是1，中 d6d0
    • 还有其他语言 Shift_JIS。。。

    – 发展到这里ASC||明显不够用了

    为了解决各个国家编码不一致的情况，因为总有需要在一个页面显示多种字符的情况。
    全球统一编码： unicode出现了

    • 全球所有文字都有唯一编码
    • 一个unicode字符通常占用2个字节
    • JAVA使用unicode编码

    为什么需要UTF-8编码？

    • 英文unicode编码和ASC||不一致
    • 包含大量英文的文本会浪费空间
    • UTF-8是变长的编码
    • 英文的UTF-8编码和ASC码是一致
    • 使用UTF编码UNICODE字符需要2-6个字节不等
    • UTF-8编码容错能力强

    编码最佳实践

    • java 使用unicode编码
    • java 程序运行使用unicode编码
    • 输入输出时，把String 和 byte[]转换，需要考虑编码
    • 始终优先考虑UTF-8
