    IDE

    • IDE 是程序员的武器，用于快速生成代码环境。
    • 自动整理代码格式。
    • 进行工程调试。

    流行

    流行的IDE是免费Eclipse(IBM开发捐赠给开源社区) 和 商用的IntelliJ Idea，如果有edu邮箱可以使用商用版本的，体验更好。

    eclipse 配置

    • 下载 eclipse for java 安装程序。

    • 安装完毕后，配置编码规则，打开eclipse，进入window-preference-general-workspace,修改 text file encoding 为UTF-8，new text file line delimiter 为Unix(window换行是\r\n,Unix是\n)

    • 调节字体

      打开eclipse，点击window，选择preferences。
      鼠标双击General，并依次打开Appearance/Colors and fonts/Basic。
      点击Basic下的Text Font。
      点击“Edit”。
      选择合适的字体大小，点击确定就OK了。

    • 快捷键

      快捷键 说明
      Ctrl+Shift+F 约定格式刷
      F6 单步调式
