    第一个JAVA

    • 新建一个Hello.java文件，输入以下代码：

    • class 的修饰符 public 是为了支持在命令行运行;也就是说不在命令行运行的话，可以省略。

    • class后面跟的是类名，类名必须和文件名一致。

    • Java 是大小写敏感的。

    • java源码要使用javac编程成字节码，然后java 调用 JVM运行字节码。

      环境部署

      下载JDK

      到oracle 官方网站下载JAVA SE JDK，根据不同操作系统，选择不同版本的JDK
      现在2020年4月，最新版本的JDK是是JAVA 14

      安装

      在window环境下，直接双击按转包，一路点下去完成安装。
      安装完成，在window下，启动命令行 window+R

      配置环境变量

    • 为什么要配置环境变量？
      为了在任何目录下都可以找到JAVA的运行环境或JAVA文件的编译器程序。多说一句，如果要运行程序，需要找到程序的位置；操作系统在当前目录下找不到运行的程序，会去环境变量中寻找；环境变量中也没有的话，就会报错。所以配置环境变量是为了全局（在宿主环境的任何位置）运行。

    • 配置方法：

      1. window->控制面板->系统->高级系统设置->环境变量->Path->编辑 ：将JAVA 安装目录下的 bin 地址写入环境变量。然后将JAVA环境变量顺序放在最上方。
      2. OS 略

    安装成功

    • 在window 下，使用 window+R 组合键打开命令行 执行命令：

      java -version //返回JAVA版本信息
      javac //不报错

    JDK 安装程序下的bin 目录

    • Java 这个执行文件就是JVM，运行java 就是启动JVM，让JVM执行java字节码(.class文件)。
    • javac java编译器，运行javac可以将java源码转换为java字节码。
    • jar 将一组class 文件打包成一个.jar文件，方便发布。（类似zip）
    • javadoc 将特定的注释文件直接生成文档。
    • jdb 在开发的过程中，进行调试。
