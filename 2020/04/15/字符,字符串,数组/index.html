    字符类型

    character
    字符类型是基本类型:

    • char:保存一个字符，可以是一个英文或中文字符。
      Java 使用Unicode 表示字符
      如果我们想看一个字符的的unicode编码，我们可以把一个char类型值，赋给int类型变量。
    • 我们也可以直接用Unicode编码表示一个字符,。

    引用类型

    引用类型的值存在堆内存中。CPU如果想获取引用类型的值，首先获得变量，然后变量会保存一个堆内存地址，然后CPU获得该地址，从这个地址中取得信息。
    堆内存数据，需要两步才可以获得。
    引用类型的变量，是指向某个对象；并不是持有某个对象。

    • 引用类型不会改变，只是引用对象的地址不会改变。

      字符串类型

      字符串类型是引用类型：

      • String:保存一个字符串
      • 转义字符\
      • 常见的转义字符：\n \r \t

    字符串操作

    • 字符串 连接 +
    • 可以连接字符串和其他数据类型

      空值null

    • String s=null; 当一个String 变量被赋值null,这个含义是不指向任何对象或内存地址。
    • String s1=””; 字符串 s1 指向一个地址或者对象，这个地址中存储的值是空值。

    数组类型

    • 当我们有一组类型相同的变量时，可以用数组表示。
    • 数组是引用类型.
    • 数组的元素可以是值(int[])类型或者引用类型(String[])
    数组类型
    定义方式 ：类型关键字[] 名称
    初始化：new int[5]；5代表是5个元素，也就是数组长度
    所有元素初始化为默认值
    索引从0开始,通过索引（下标）来访问元素，如果访问的索引不存在，也就是超出数组范围，运行时报错
    创建后元素个数不能改变，length属性存储着元素个数
    • 指定初始化元素
