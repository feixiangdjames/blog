    #IO
    IO是指Input/Output,即输入和输出：

    • Input是指外部读入数据到内存，例如，读文件，从网络读取等
    • Outpu指吧数据从内存输出到外部，例如：写文件，输出到网络等

    IO Stream

    IO 流是一种顺序读写数据的模式

    • 单向流动
    • 以Byte为最小单位

    如果字符不是单字节(byte)表示的ASCII:

    • JAVA 提供了 Reader/ Writer表示字符流
    • 字符流传输的最小数据单位是char
    • 字符流输出byte取决于编码方式

    Reader/Writer本质上是一个能自动编解码的InputStream/OutputStream

    • 使用Reader:

      虽然数据源是byte，我们读取数据都是char类型的字符。因为Reader内部，把
      读入的byte做了编码.

    • 使用InputStream:

    使用InputStream,我们读入的数据和原始二进制是一样的，都是byte数组。
    但是，我们自己可以把byte数组，按照某种编码方式，进行转换，达到和reader一样的效果。

    究竟使用Reader 还是 InputStream 要看具体的使用场景。

    • 如果数据源不是文本（txt）,那么只能使用inputStream.
    • 如果数据源是文本，使用Reader就更方便一些。

    Writer 和 OutputStream 类似的。

    同步和异步

    同步IO：

    • 读写IO时，代码等待数据返回才执行后续代码。
    • 代码编写简单，CPU执行效率低。

    异步IO：

    • 读写IO时，仅发出请求，然后立即执行后续代码。
    • 代码编写复杂，CPU执行效率高。

    JDK提供的java.io是同步IO;java.nio是异步IO

    java.io

    抽象类 实现类
    InputSteam FileInputStream
    OutputStream FileOutputStream
    Reader FileReader
    Writer FileWriter
    总结
    IO流是一种流式数据输入/输出模型
    二级制数据以byte为最小单位在InputStream/OutputStream单项流动
    字符数据以char为最小单位在Reader/Writer中单向流动
    JDK的java.io包提供了同步IO功能
    Java的IO流的接口和实现式分类的：
    字节流接口：inputStream/OutputStream
    字符流接口：Reader/Writer

    File

    java.io.File表示文件系统的一个文件或者目录：

    • 构造方法:File(String pathname)
      //Windows:
      File f=new File(“c:\windows\notepad.exe”)

    //Linux:
    File f=new File(“usr/bin/javac”)

    //相对路径：
    假设当当前目录是C:\docs
    File f3= new File(“.\sub\javac”);//绝对路径是C:\docs\sub\javac
    File f4= new File(“..\sub\javac”);//绝对路径是C:\docs\javac

    file对象有3个路径方法：

    file 对象判断一个对象是不是文件或者目录：

    • boolean isFile():是否是文件
    • boolean isDirectory():是否是目录

    当File对象f，f.isFile()==true，可以如下方法：

    • boolean canRead()
    • boolean canWrite()
    • boolean canExcute():是否允许运行
    • long length():获取文件大小
    • boolean createNewFile():创建一个新文件
    • static boolean createTempFile(String prefix,String suffix):创建一个临时文件
    • boolean delete():删除该文件
    • void deleteOnExit():在JVM退出时删除该文件

    当File对象表示一个目录是（isDirectory()==true）:

    • String[] list()
    • File[] listFiles()//列出文件和目录
    • File[] listFiles(FileFilter filter)
    • File[] listFiles(FilenameFilter filter)
    • boolean mkdir() : 创建该目录
    • boolean mkdirs():创建该目录，并在必要时将不存在的父目录也创建出来
    • booean delete():删除该目录
