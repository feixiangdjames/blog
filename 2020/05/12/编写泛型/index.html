    1. 编写泛型类

    编写泛型，需要定义泛型类型
    静态方法不能引用泛型类型,必须定义其他类型来实现泛型
    public static Pair creat(K frist,K end )
    泛型可以同时定义多种类型<T,K>

    编写泛型类

    • 编写泛型类比普通类复杂
    • 泛型类一般用在集合类中ArrayList
    • 很少需要编写泛型类

    泛型类型不能用于静态方法：

    • 编译错误
    • 编译器无法在静态字段或静态方法中使用泛型类型

    静态方法可以单独改写为“泛型”方法：

    • 使用另一个类型

    泛型可以支持多种类型：

    public class Pair<T,K>{…}
